BeepBot

The purpose of this project is to design a simple yet effective proximity alarm using an Arduino microcontroller, an ultrasonic distance sensor, an LED, and a buzzer. The system works by measuring the distance between the sensor and a nearby object. When the object moves within a certain range, the LED lights up and the buzzer activates to signal that the object is too close. This project demonstrates basic principles of electronics, sensor integration, and microcontroller programming. To enhance functionality, the alarm could be expanded to include distance-based responses, such as varying buzzer tones or LED brightness depending on proximity.

The Blinker

The purpose of this project is to design a reliable LED blinker system using an Arduino microcontroller and discrete electronic components. The Arduino is connected to the LED through appropriate circuitry and programmed to control precise on and off timing using embedded delay logic. This project demonstrates core principles of microcontroller I/O control, hardware–software integration, and basic embedded system design, and it can be extended to support adjustable blink rates or external input control.

The Last Flower

The Last Flower is a 3D narrative game developed in Unreal Engine 5 with gameplay systems programmed in C++. The project emphasizes environmental storytelling and player immersion through dynamic worldbuilding. I created a responsive weather system with rain, lightning, and falling leaves, developed AI with chase and flying behaviors, and designed the title screen and pause/play menus for smooth, intuitive interaction. Visual assets were refined in Photoshop to maintain a consistent aesthetic. The game was successfully demoed and published, showcasing teamwork, technical skill, and project completion.

ColorBlind Exam

This project was my first coding experience using Python. I developed a program to simulate and test color blindness, allowing users to explore how different color vision deficiencies affect perception. Through this project, I applied basic programming concepts such as variables, loops, conditional statements, and user input handling. Completing it helped me build foundational coding skills, practice logical problem-solving, and gain confidence in writing and debugging Python code.
